JVM(Java Virtual Machine)
	JAVA 프로그램을 실행해 준다.
	(JAVA가 꽃이라면 JVM은 땅)

JRE(Java Runtime Environment)
	JVM을 생성하는 부분
	JVM에서 실행할 떄 필요한 라이브러리 파일들과
	기타 파일들을 가지고 있다.
	(산소+햇빛+물+...)

JDK(Java Development Kit) 자바 개발 도구
	JVM과 JRE에 의해 실행되고 구동될 수 있는
	자바 프로그래밍을 생성할 수 있게 해준다.
	JRE 외에 개발에 필요한 도구들을 가지고 있다.
	(원소,자원 같은 개념)

JDK 설치
	구글 jdk 8 검색 -> 첫번째 링크 -> 자기에 맞는 운영체제 선택
	-> 다운받은 파일 실행 -> 설치 경로에서 Program Files 제거
	-> next -> JRE 설치 창 -> JRE 경로 앞에서 설치한 JDK 경로
	찾아가기 -> jdk 폴더와 동일한 위치에 JDK만 jre로 바꾸어서
	폴더 생성 -> 그 폴더 선택한 후 설치
