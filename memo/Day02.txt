값
	데이터 그 자체
	문자열, 숫자, 그림, 영상, ...

byte는 데이터의 최소단위(8bit)-의미를 갖는 최소 단위
bit는 컴퓨터 용량의 최소단위(1,0)

값의 종류(데이터 타입)

		type	byte	값

		short	2
	정수형	int(기본)	4	10, -12, 0, ...
		long	8

	실수형	float	4	10.248, -2.187
	      double(기본)	8	10.248, -2.187

	문자열 	String	??	"abc", "0", "a"

	>>RAM에 x라는 이름의 저장공간이 할당되고(생성되고),

주소(c언어)
	저장 공간이 RAM에 할당되면 고유한 주소값이 부여된다.

변수 선언시 주의사항
	같은 이름의 변수를 두개 이상 만들 수 없다.
	되도록 선언부에 한꺼번에 선언한다.(메소드의 상단부분)
	초기화를 해준다.

변수명 주의사항
	숫자는 맨 앞에 쓸 수 없다.
	특수문자는 쓸 수 없다.
	단, 언더바(_)는 사용 가능하다.
	되도록 소문자만 사용한다.(대문자는 class)
	공백을 사용할 수 없다.
	명사형을 사용한다.
	되도록 한글은 사용하지 않는다.
	뜻이 있는 단어를 사용한다.
	-a,b,c,d(X) -num,height, score, cnt(O)

변수의 사용

	2. 값에 의미를 부여하기 위해서

서식 문자(format문자)
	따옴표나 쌍따옴표 안에서 작성

	"%d" : 정수 출력
	"%s" : 문자열 출력
	"%f" : 실수 출력
	"%c" : 문자 출력
	"%%" : % 출력

형변환
	값의 형태를 바꾸어 주는 일

	자동형변환
		정수 + 실수 = 실수
		문자열 + 숫자(정수,실수) = 문자열
		문자 + 정수 = 정수(아스키코드)

	강제형변환(Casting)
		(double)10
		(자료형)값

		(자료형) : 형변환연산자

입력 메소드
	입력 상태 : 커서가 깜빡이며 값을 입력하기를 기다리는 상태
	개발자가 할수 있는건, 입력상태로 만들어주는것 까지이다.
	입력 받기 전에 사용자가 알맞은걸 입력할 수 있도록 먼저 출력을 해준다.
	
	new Scanner(System.in).next();
		↓
	Scanner sc = new Scanner(System.in); - 입력을 받기 위한 코드
	String msg = sc.next

	next()	: 엔터전까지의 통채로가 사용자가 입력한 문자열 값이다.
		정 회윤 : "정"과 "회윤"으로 분리해서 앞에것만 받는다.
		즉, 띄어쓰기 기준으로도 뒤에값들은 잘라낸다.
		
	nextLine()	: 통채로가 사용자가 입력한 문자열 값이다.
		띄어쓰기도 문자열에 포함해서 입력 받는다.
		정 회윤 : "정" "회윤"
		nextLine()은 앞에 입력을 했었다면, 그 입력코드에서 
		마지막에 쓴 Enter를 받아버리므로, 
		중간에 의미없는 nextLine()하나를 추가해 주어야 한다.

	nextInt()	: 통채로가 사용자가 입력한 정수값이다.
