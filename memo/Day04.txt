조건문
	조건을 제시하여 흐름을 나눠주는 역할
	조건에 따라 경우의 수를 나누기
	특정한 상태 물어볼때 사용(num가 5니? / 비밀번호가 이거랑 같니?)

	if ~ else if ~ else
	
		if(조건식1){
			조건식1이 참이면 수행
		}else if(조건식2){
			조건식2가 참이면 수행
			(조건식 1은 거짓이다)
		}...
		else{
			위의 조건식들이 모두 거짓이면 수행
		}

		조건식 중에 하나가 참일 경우, 안의 내용을 실행한 후
		아래의 모든 else if, else문을 skip한다.
조건식이 반복될 때, 값이 많이 나올때
	switch ~ case ~ default		

		switch(변수){
		case 값1:
			변수==값1일때 수행
			break;
		case 값2:
			변수==값2일때 수행
			break;
		case 값3:
			변수==값3일때 수행
			break;
		...
		default:
			위의 경우에 들어가지 않을 때 수행
		}

case하나에 들어가면 그밑에 다른 케이스들이 수행 되기 때문에 break를 쓴다.
break
	흐름을 탈줄할 때 사용

while문
	while(조건식){
		조건식이 참일동안 수행
	}
초기화로 인한 초기값이 애초에 while문이 거짓일 경우
최초 한번을 실행하기 위함
do~while문
	do{
		조건식이 참일동안 수행
	}while(조건식);

요즘에는 속도나 메모리가 여유가 많이있기때문에 
while(true){

   }
}
와 같은 형태로 비교를 두번하는 형태로 만든다.

	}

증감연산자(두가지)
	i++ : i 1 증가
	i-- : i 1 감소

	전위형
		i++;
		처리를 먼저하고 증감을 해준다
	후위형
		++i;
		증감을 먼저하고 처리를 해준다
